# 🛍️ Digital Product Store

A modern React-based eCommerce UI project with cart functionality and clean UI.

---

## 🚀 Features
- Product listing (3 column layout)
- Add to cart system
- Remove from cart
- Cart total counter in navbar
- Toggle between products & cart view
- Toast notifications for actions
- Responsive design
- Checkout functionality (clears cart)

---

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- React Toastify
- JSON Data

---

## 📦 Features Breakdown
- Products displayed dynamically from JSON
- Cart system with state management
- Prevent duplicate items in cart
- Real-time cart updates
- Mobile responsive UI

---

## 📸 UI Sections
- Navbar
- Banner
- Stats
- Product Grid
- Cart Page
- Steps Section
- Pricing Section
- CTA
- Footer
